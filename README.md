# 🌸 サポートカード(ローカル高セキュリティ版)

重症心身障がいのあるお子さんの特徴を、初めて会う支援者・先生・医療者に「必要な情報だけサッと」伝えるためのアプリです。知りたいトピックをタップすると、その情報だけが大きな文字で表示されます。

**このバージョンはサーバー不要。データはすべてお使いの端末内だけに、暗号化して保存されます。**

## ✨ 機能
- 🏠 チーム制(「家族」「学校」「夏のキャンプ」など目的別にカードをまとめる)
- 👥 カードの作成・編集・削除/トピックの追加・削除・並べ替え/写真
- 🚨 緊急情報の優先表示/🖨 印刷・PDF/🔊 読み上げ/🔤 文字サイズ/🎨 テーマ4色
- 📤 チームの共有(暗号化ファイル .scard を書き出し→相手が読み込み)

## 🔐 セキュリティ
- **端末内のみ保存・外部通信ゼロ**(CSPでブラウザレベルに遮断)
- **AES-256-GCM暗号化**(鍵はパスコードからPBKDF2/SHA-256・31万回で導出)
- **総当たり対策**:連続で間違えると待機時間が延びる
- **自動ロック**:15分操作がないとロック画面へ
- **覗き見ガード**:アプリが背面に回ると画面を隠して自動ロック
- XSS対策・noindex

> ⚠️ パスコードを忘れるとデータは復元できません。必ず控えてください。
> ⚠️ データは端末ごと。機種変更やブラウザのデータ消去で消えます。定期的に「共有」で書き出してバックアップを。

## 🚀 使い方
### PCですぐ試す
3ファイル(index.html / style.css / app.js)を同じフォルダに置き、index.html をダブルクリック。

### GitHub Pagesで公開(スマホでも使える)
1. GitHubでリポジトリを作成し、この3ファイル+下記をアップロード
2. Settings → Pages → Branch「main」→ Save
3. 発行されたURLをスマホで開き「ホーム画面に追加」

初回はパスコードを設定 → ホームでチーム作成 → カード追加、の順です。

## 🛠 ファイル構成
```
index.html   ページ骨組み(セキュリティ設定もここ)
style.css    見た目(冒頭のデザイントークンで色を一括変更)
app.js       アプリ本体・暗号化・自動ロック
test.mjs     自動テスト(node test.mjs)
README.md / PRIVACY.md / LICENSE
```
