# サポートカード(ローカル高セキュリティ版)

障がいのあるお子さんの特徴を、**必要な情報だけサッと伝えられる**プロフィールカードアプリです。
初めて会う支援者・先生・医療スタッフに、「この子はこういう子です」を一目で伝えられます。発作時の対応、食事の形態、コミュニケーションのサイン、好きなこと——知りたいトピックをタップする
その情報だけが大きな文字で表示されます。
**このバージョンは「チーム」単位で共有先を分けられます。**「家族」「学校」「夏のキャンプ」など、伝える相手ごとにチームを作り、それぞれ別のメンバー・別のパスコードで運用できます。イベントが終わったらチームごと消せば、渡した情報を回収できます。また、サーバー不要で、データはすべてお使いの端末内だけに、暗号化して保存されます。


## 機能
- チーム制(「家族」「学校」「夏のキャンプ」など目的別にカードをまとめる)
- カードの作成・編集・削除/トピックの追加・削除・並べ替え/写真
- 緊急情報の優先表示/印刷・PDF/読み上げ/文字サイズ/テーマ4色
- チームの共有(暗号化ファイル .scard を書き出し→相手が読み込み)


## セキュリティ
- **端末内のみ保存・外部通信ゼロ**(CSPでブラウザレベルに遮断)
- **AES-256-GCM暗号化**(鍵はパスコードからPBKDF2/SHA-256・31万回で導出)
- **総当たり対策**:連続で間違えると待機時間が延びる
- **自動ロック**:15分操作がないとロック画面へ
- **覗き見ガード**:アプリが背面に回ると画面を隠して自動ロック
- XSS対策・noindex

- **パスコード①:端末のパスコード**
- 端末の中にある全データ(全チーム・全カード)は、このパスコードで暗号化されて保存されています
- アプリを開くたびに入力します(15分放置や背面移行で再度必要に)
- 端末ごとそれぞれ別のアプリ・別のデータ・別のパスコードを持ちます（同期しない）

- **パスコード②:共有パスコード(チームを渡すときに、その場で決めるもの)**
- 以下の流れでデータは共有できる：
- 「📤 このチームを共有」を押す
- その場で共有用のパスコードを決める(例:hana2025)。①とは無関係の、好きな合言葉でOK
- 「やまだ家.scard」という暗号化されたファイルが書き出される。中身はこの共有パスコードで鍵がかかっています
- そのファイルを相手に渡す。パスコードは別の方法で伝える(ファイルと別ルートが安全)
-
- 相手は自分の端末で「📥 読み込み」→ もらった.scardファイルを選ぶ
- 共有パスコード(hana2025)を入力
- 正しければ、ファイルの鍵が開いて、そのチームが相手のアプリに新しいチームとして追加される
- 以後、それは相手の端末のデータになり、相手の端末パスコード①で守られます

> パスコードを忘れるとデータは復元できません。必ず控えてください。
> データは端末ごと。機種変更やブラウザのデータ消去で消えます。定期的に「共有」で書き出してバックアップを。


## はじめかた

初回はパスコードを設定 → ホームでチーム作成 → カード追加、の順です。
