# サポートカード(ローカル高セキュリティ版)

障がいのあるお子さんの特徴を、**必要な情報だけサッと伝えられる**プロフィールカードアプリです。
初めて会う支援者・先生・医療スタッフに、「この子はこういう子です」を一目で伝えられます。発作時の対応、食事の形態、コミュニケーションのサイン、好きなこと——知りたいトピックをタップする
その情報だけが大きな文字で表示されます。
**このバージョンは「チーム」単位で共有先を分けられます。**「家族」「学校」「夏のキャンプ」など、伝える相手ごとにチームを作り、それぞれ別のメンバー・別のパスコードで運用できます。イベントが終わったらチームごと消せば、渡した情報を回収できます。また、サーバー不要で、データはすべてお使いの端末内だけに、暗号化して保存されます。


## 機能
- チーム制(「家族」「学校」「夏のキャンプ」など目的別にカードをまとめる)
- カードの作成・編集・削除/トピックの追加・削除・並べ替え/写真
- 緊急情報の優先表示/印刷・PDF/読み上げ/文字サイズ/テーマ4色
- チームの共有(暗号化ファイル .scard を書き出し→相手が読み込み)


## セキュリティのしくみ

### 基本の考え方
データは**インターネットに送られません**。すべてあなたの端末の中だけに、暗号化して保存されます(外部通信は技術的に遮断済み)。だから、サーバーから情報が洩れる心配がありません。

### 2種類のパスコード
アプリには、役割の違う2つのパスコードがあります。

#### ① 端末のパスコード（アプリを開く鍵）
- 最初の起動時に、あなたが決めます
- 端末の中の全データ（すべてのチーム・カード）が、このパスコードで暗号化されます
- アプリを開くたびに入力します（15分の放置や、他アプリに切り替えた後も再入力）
- **端末ごとに別々**です。あなたのスマホと家族のスマホは、それぞれ独立したデータとパスコードを持ちます（自動では同期しません）


#### ② 共有パスコード（チームを渡すときの鍵）
①とは別物で、誰かにチームを渡すときに、その場で決める合言葉です。
**渡す側の手順**
1. チーム画面で「📤 このチームを共有」を押す
2. 共有用のパスコードをその場で決める（例：`hana2025`）。①とは無関係でOK
3. 暗号化されたファイル（例：`やまだ家.scard`）が書き出される
4. ファイルを相手に渡し、**パスコードは別の方法で伝える**

> ファイルとパスコードを別ルートで渡すのが安全です。両方が同じ経路で漏れなければ、中身は守られます。


**受け取る側の手順**
5. 「📥 読み込み」から、もらったファイルを選ぶ
6. 共有パスコードを入力する
7. 正しければ、そのチームが**新しいチームとして自分のアプリに追加**される
8. 以後そのデータは相手の端末のものになり、相手の①のパスコードで守られます

> これは「コピー」です。渡したあとに自分が情報を更新しても、相手の手元は古いままなので、最新を渡すには共有し直してください。

### 端末を守る4つの仕組み


| 仕組み | 内容 |
|---|---|
| **暗号化** | AES-256-GCM（鍵はパスコードから31万回の計算で生成。パスコード自体は保存されません） |
| **総当たり対策** | パスコードを連続で間違えると、待ち時間が少しずつ延びます |
| **自動ロック** | 15分操作がないと、自動でロック画面に戻ります |
| **覗き見ガード** | 他のアプリに切り替えると画面が隠れ、自動でロックされます |

そのほか、入力内容のエスケープ処理（XSS対策）、検索エンジン非掲載（noindex）にも対応しています。


### ⚠️ 必ず守ってほしいこと
- **パスコードを忘れると、データは誰にも復元できません。** 必ず控えてください（本物の暗号化のため、開発者でも開けられません）。
- **データは端末ごとに保存されます。** 機種変更やブラウザのデータ消去で消えます。「📤 共有」での書き出しを、定期的なバックアップとして使ってください。
