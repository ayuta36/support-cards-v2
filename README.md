# 🌸 サポートカード

**この子のこと、必要な情報だけサッと伝わる。**
障がいのあるお子さんの特徴を、初めて会う支援者・先生・医療スタッフに一目で伝えるためのプロフィールカードアプリです。

</div>
---

## 概要
発作時の対応、食事の形態、コミュニケーションのサイン、好きなこと——知りたいトピックをタップするだけで、その情報だけが大きな文字で表示されます。
「家族」「学校」「夏のキャンプ」など、**伝える相手ごとに「チーム」を分けて**運用できます。イベントが終わったらチームごと削除すれば、渡した情報を回収することも可能です。
データはすべてお使いの端末内だけに暗号化して保存され、インターネットには送信されません。

---

## 主な機能

| | 機能 |
|---|---|
| **チーム制** — 目的別にカードをまとめる(家族／学校／イベント など) |
| カードの作成・編集・削除、トピックの追加・削除・並べ替え、写真の登録 |
| 緊急情報の優先表示 |
| 印刷・PDF出力 / 音声読み上げ / 文字サイズ3段階 / テーマ4色 |
| チームの共有(暗号化ファイル `.scard` を書き出し → 相手が読み込み) |

## セキュリティのしくみ

### 基本の考え方

データは **インターネットに送られません。** すべてあなたの端末の中だけに暗号化して保存されるため、サーバーから情報が漏れる心配がありません。


### 2種類のパスコード

役割の異なる2つのパスコードがあります。

#### ① 端末のパスコード(アプリを開く鍵)

- 最初の起動時に、決めます
- 端末内の全データ(すべてのチーム・カード)が、このパスコードで暗号化されます
- アプリを開くたびに入力します(15分の放置や、他アプリへの切り替え後も再入力)
- **端末ごとに別々です。** あなたのスマホと家族のスマホは、それぞれ独立したデータとパスコードを持ちます(自動同期はしません)


#### ② 共有パスコード(チームを渡すときの鍵)

①とは別物で、誰かにチームを渡すときに、その場で決める合言葉です。

<table>
<tr><td>
  
**渡す側**

1. チーム画面で「📤 このチームを共有」を押す
2. 共有用パスコードをその場で決める(例:`hana2025`)
3. 暗号化ファイル(例:`やまだ家.scard`)が書き出される
4. ファイルを相手に渡し、**パスコードは別の方法で伝える**

</td><td>

**受け取る側**

5. 「📥 読み込み」からファイルを選ぶ
6. 共有パスコードを入力する
7. 正しければ、**新しいチームとして追加**される
8. 以後そのデータは相手の端末のものになり、相手の①で守られる

</td></tr>
</table>

> **ファイルとパスコードは別ルートで渡す**のが安全です(例:ファイルはメール、パスコードは口頭)。両方が同じ経路で漏れなければ中身は守られます。
> これは「コピー」です。渡したあとに自分が更新しても相手の手元は古いままなので、最新を渡すには共有し直してください。

### 端末を守る4つの仕組み

| 仕組み | 内容 |
|---|---|
| **暗号化** | AES-256-GCM(鍵はパスコードから PBKDF2/SHA-256・31万回 で生成。パスコード自体は保存されません) |
| **総当たり対策** | パスコードを連続で間違えると、待ち時間が少しずつ延びます |
| **自動ロック** | 15分操作がないと、自動でロック画面に戻ります |
| **覗き見ガード** | 他アプリに切り替えると画面が隠れ、自動でロックされます |


そのほか、入力内容のエスケープ処理(XSS対策)、検索エンジン非掲載(noindex)にも対応しています。


### 必ず守ってほしいこと

- **パスコードを忘れると、データは誰にも復元できません。** 必ず控えてください(本物の暗号化のため、開発者でも開けられません)。
- **データは端末ごとに保存されます。** 機種変更やブラウザのデータ消去で消えます。「📤 共有」での書き出しを、定期的なバックアップとしてご利用ください。

## はじめかた

## ファイル構成

```text
├── index.html    ページの骨組み(セキュリティ設定も含む)
├── style.css     見た目(冒頭の「デザイントークン」で色を一括変更可)
├── app.js        アプリの動作・自動ロック・暗号化
├── test.mjs      自動テスト(node test.mjs で実行)
├── PRIVACY.md    プライバシーポリシー・利用規約
├── README.md     本ファイル
└── LICENSE       MIT License
```

フレームワークやビルドツールは使用していません。3ファイル(index.html / style.css / app.js)だけで動く、依存ゼロの構成です。
